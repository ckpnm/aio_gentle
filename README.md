
# A I O - GENTLE  UTILITY

Интерактивная TUI-утилита для изящной, быстрой и модульной настройки сервера под VPN (Remnawave / Xray), комплексной защиты сети и глубокой диагностики. 

Скрипт отличается минималистичным визуальным оформлением, плавными анимациями прогресса и абсолютной независимостью модулей — вы сами выбираете, что именно нужно установить на сервер в данный момент.

## ⚡️ Быстрый старт

Запустите скрипт одной командой от пользователя `root`:

```bash
bash <(curl -sL [https://raw.githubusercontent.com/ckpnm/aio_gentle/main/main.sh](https://raw.githubusercontent.com/ckpnm/aio_gentle/main/main.sh))```


🛠 Возможности
Базовая подготовка и сеть
Базовые зависимости: Установка необходимых системных утилит за один клик.

Сетевой тюнинг: Включение TCP BBR для ускорения сети и полное отключение IPv6 (если требуется).

Безопасность SSH: Интерактивное изменение стандартного порта SSH.

Развертывание VPN (Remnawave)
Среда: Автоматическая установка Docker и Docker Compose.

Сертификаты: Запуск генерации сертификатов и маскировки через Caddy Selfsteal.

Нода: Интерактивное развертывание контейнера remnanode с привязкой вашего SecretKey.

Логирование: Настройка автоматической ротации и сжатия логов Xray (logrotate), чтобы сервер не задохнулся от нехватки памяти.

Защита и Фильтрация
Базовая защита: Настройка UFW (файрвол) и автоматическая установка Fail2Ban.

Traffic Guard: Установка защиты от сканеров и ботов.

AS Block: Атомарная блокировка "грязных" подсетей Leaseweb и Hurricane Electric (AS6939+) через ipset с добавлением еженедельного автообновления баз в Cron.

URL Block: Гибкая блокировка по кастомным спискам (URL) с использованием nftables и systemd-таймеров.

Cloudflare WARP: Установка нативного клиента WARP.

Диагностика и Инфо
Reality Keys: Быстрый вывод ключей x25519 из активного контейнера и данных для подключения ноды к панели.

IP Region Check: Мощный сканер гео-принадлежности IP-адреса по базам GeoIP (MaxMind, RIPE и др.) и стриминговым/популярным сервисам (Netflix, Spotify, ChatGPT и др.).

Speedtest (Ookla): Установка и нативный запуск официального CLI спидтеста для замера скорости сервера.

⚙️ Управление скриптом
Утилита умеет обновлять саму себя (сверяя версии с GitHub) и корректно удалять свои следы из системы через меню управления. Логи всех фоновых установок аккуратно складываются в /var/log/aio_setup.log, не засоряя экран.
